Publish a standard media feed for your Lazyest Gallery

== Description ==

Media Feed plugin for [Lazyest Gallery](http://wordpress.org/extend/plugins/lazyest-gallery/)

Create Media feeds for Lazyest Gallery for use with viewers like Cooliris

Retrieve your Media feed at: 

* if you have Default permalinks: (your blog address)/?lazyestfeed=0 
* if you have Another permalink structure (your blog address)/lazyestfeed.xml 

= In version 0.2 =
* Basic Media Feed for Lazyest Gallery

== Installation ==

1. Install using the WordPress plugin installer
2. Activate
3. Copy the file crossdomain.xml to the root of your webserver
4. Retrieve your Media feed at (your blog address)/lazyestfeed.xml
